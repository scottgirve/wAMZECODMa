# 项目简介

## 核心功能
- 数据采集与预处理
- 模型训练与预测
- 结果展示与交互

## 适用场景
- 医疗领域数据预测
- 早期脑中风风险评估

## 技术特点
- 基于 Django 框架进行后端开发
- 利用 Pyecharts 实现数据可视化
- 结合随机森林和线性回归算法进行预测
